# 🗺️ CODMON - 3D İnteraktiv Taktiki Xəritə (Swarm Control)

Bu layihə, **CODMON** pilotsuz dəniz/hava dron dəstələrinin (Swarm Intelligence) qlobal əməliyyatlarını vizuallaşdırmaq üçün hazırlanmış **3D Taktiki İdarəetmə Paneli (Mission Control UI)** simulyasiyasıdır.

## 🚀 Xüsusiyyətlər
* **Three.js 3D Sahə:** Tam dinamik fırlanan, yaxınlaşan kibernetik dəniz və radar fonu.
* **Dinamik Dron Dəstəsi (Swarm):** Riyazi alqoritmlərlə bir-biri ilə koordinasiyalı və xaotik hərəkət edən 12 ədəd CODMON dron vahidi.
* **İnteraktiv İdarəetmə (Tactical Controls):** * *ENGAGE ATTACK MODE:* Dronları dərhal mərkəzdəki qırmızı düşmən hədəfinə kilidləyir və hücum radiusuna daxil edir.
    * *RE-ROUTE SWARM PATH:* Dronların uçuş marşrutunu ani olaraq yenidən hesablayır.
* **Canlı Telemetriya:** Real vaxt rejimində dron hündürlükləri, koordinatlar və kompası göstərən hərbi bunker interfeysi.

## 🛠️ Necə İşləyir?
Heç bir quraşdırmaya və ya lokal serverə ehtiyac yoxdur. HTML faylı birbaşa CDN vasitəsilə lazımi kitabxanaları yükləyir və işə salır.
